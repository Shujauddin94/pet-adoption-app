## Features

- List all pets
- Add new pets
- Edit existing pets
- Delete pets
- View pet details
- Responsive UI with Material-UI
- Works with Strapi v5 `documentId`

## Tech Stack

- **Backend:** Strapi v5 (Node.js)
- **Frontend:** React, Vite, Material-UI
- **HTTP Client:** Axios
